# 월별 공통부서 인원수 입력 가이드

## ✅ 생성된 파일

### 1. 입력 템플릿 (작성 필요)
**파일 위치:** `./out/templates/월별_공통부서_인원수_입력템플릿.xlsx`

이 파일을 열어서 인원수를 입력해주세요!

### 2. 샘플 데이터 (참고용)
**파일 위치:** `./out/templates/월별_공통부서_인원수_샘플.xlsx`

입력 형식 참고용 샘플 데이터입니다.

## 📝 입력 방법

### Step 1: 템플릿 파일 열기
```
./out/templates/월별_공통부서_인원수_입력템플릿.xlsx
```

### Step 2: 인원수 입력

| 기준년월 | 정규직인원수 | 비고 |
|---------|------------|------|
| 202401  | **150**    | (선택) |
| 202402  | **152**    | 신규입사 2명 |
| 202403  | **151**    | 퇴사 1명 |
| 202404  | **153**    | |
| ...     | ...        | |

**입력 예시:**
- 24년 1월: 150명
- 24년 2월: 152명
- 24년 3월: 151명
- ...
- 25년 10월: 145명

### Step 3: 파일 저장
입력 완료 후 엑셀 파일을 저장하세요.

### Step 4: 데이터 변환
다음 명령을 실행하여 대시보드용 형식으로 변환:

```bash
python load_manual_headcount.py
```

이 명령을 실행하면:
- ✅ 데이터 검증 (0인 값 확인)
- ✅ CSV 파일로 변환
- ✅ 피벗 테이블 생성
- ✅ 전년동월 비교 통계 출력

## 📊 출력 파일

변환 후 다음 파일이 생성됩니다:

1. **`./out/snowflake/headcount_monthly_latest.csv`**
   - 월별 인원 데이터 (원본 형태)
   - 대시보드에서 사용

2. **`./out/snowflake/headcount_pivot_by_month.csv`**
   - 월별 인원 피벗 테이블
   - 차트 표시용

## 💡 입력 팁

1. **일관성 유지**: 매월 말일 기준 또는 매월 1일 기준 등 일관된 기준 사용
2. **비고 활용**: 인원 변동 사유를 간단히 메모 (선택사항)
3. **검증**: 입력 후 전년동월 비교로 이상치 확인

## 🔄 데이터 업데이트

인원수가 변경되면:
1. 템플릿 파일 수정
2. `python load_manual_headcount.py` 재실행
3. 대시보드 새로고침

## 📈 활용 방안

대시보드에서:
- **1인당 비용** = 총 비용 / 인원수
- **인건비 비율** = 인건비 / 총 비용
- **인원 증감 추이** 차트
- **전년동월 인원 비교**
