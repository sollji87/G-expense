```
텍스트(적요), 거래처명, 공급업체, 금액_정제, 차변/대변지시자
```

## 💡 활용 방안

### 1. 대시보드 메인 화면
- `pivot_by_gl_yyyymm_combined.csv` 사용
- 계정별 당년/전년 비교
- 월별 트렌드 차트

### 2. 드릴다운 기능
- 계정 클릭 → `pivot_by_gl_cctr_yyyymm_combined.csv`에서 해당 계정의 코스트센터별 데이터 표시

### 3. AI 세부 분석 (24년 10월 vs 25년 10월)
- `detail_202410_all.csv`, `detail_202510_all.csv` 사용
- 적요(텍스트) 분석으로 비용 증감 원인 파악
- 예시:
  - **인건비 증감**: 적요에서 "급여", "제수당" 키워드 분석
  - **IT수수료 증감**: "IT사용료", "유지보수" 등의 거래처명/적요 분석
  - **지급수수료 증감**: "법률자문료", "컨설팅" 등 세부 항목 분석

### 4. 계정별 상세 조회
- 특정 계정 클릭 → 해당 계정의 CSV 파일 읽기
- 예: "급료와임금" 클릭 → `GL_53010101_급료와임금.csv` 표시
